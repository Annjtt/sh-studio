# Основные команды Git

## Настройка и инициализация
- git config --global user.name "Ваше Имя" - Установка имени пользователя
- git config --global user.email "ваш@email.com" - Установка email
- git init - Инициализация нового репозитория
- git clone [url] - Клонирование существующего репозитория

## Основные операции
- git status - Проверка состояния файлов
- git add [файл] - Добавление файла в индекс
- git add . - Добавление всех измененных файлов
- git commit -m "сообщение" - Создание коммита с сообщением
- git commit --amend - Изменение последнего коммита

## Работа с ветками
- git branch - Просмотр списка веток
- git branch [имя] - Создание новой ветки
- git checkout [ветка] - Переключение на ветку
- git checkout -b [имя] - Создание и переключение на новую ветку
- git merge [ветка] - Слияние веток

## Синхронизация с удаленным репозиторием
- git remote add origin [url] - Добавление удаленного репозитория
- git push origin [ветка] - Отправка изменений
- git pull origin [ветка] - Получение изменений
- git fetch - Получение изменений без слияния

## Просмотр информации
- git log - История коммитов
- git log --oneline - Краткая история коммитов
- git diff - Просмотр изменений в файлах
- git show [коммит] - Просмотр определенного коммита

## Отмена изменений
- git reset [файл] - Отмена индексации файла
- git reset --hard HEAD - Отмена всех локальных изменений
- git revert [коммит] - Создание нового коммита, отменяющего изменения

## Работа с тегами
- git tag - Просмотр списка тегов
- git tag [имя] - Создание нового тега
- git push origin [тег] - Отправка тега в удаленный репозиторий

## Временное сохранение
- git stash - Временное сохранение изменений
- git stash list - Просмотр списка сохранений
- git stash apply - Применение последнего сохранения
- git stash drop - Удаление последнего сохранения
